Exercise 1 (JavaScript)

  1. Create a service person using the module pattern.
  2. The person service keeps its details including firstName and lastName private.
  3. The person service exposes the getFirstName method.
  4. Define a controller HelloCntl using a constructor function passing it the person service.
  5. The HelloCntl controller exposes the sayHello method which returns a greeting (as as a string) getting the name from the person service.

Exercise 2 (Controller, Service)

  • Create Angular service which exposes method getTodos which returns list of following objects 
    
{
    title: 'example title',
    description: 'example description',
    done: false
}
  • Create Angular Controller and use created service
  • Bind result of getTodos to controller variable and display using ng-repeat

Exercise 3 (Routing)

  • Install angular-router using bower
  • Extend objects from getTodos method by id property
  • Create list and detailed view for objects from getTodos method using ng-router
  • List view should display simple list of objects returned by service(#/list)
  • Detailed view should display details of one item(#/item/1)

Exercise 4 (Backend)

  • Replace existing service with real backend service
  • Add persistance(saveTodos)
  • Add ability to create new Todos(be creative ;))
  • Add archiving functionality
