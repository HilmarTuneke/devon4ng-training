Details

  • Uses TCP/IP
  • Can be any port (not only :80)
  • Version 1.1 is mostly used
  • Stateless
  • Max 2 connections per browser to server

HTTPS

  • You should always use HTTPS.
  • It is generally transparent to application.
  • HTTPS is limited to one domain - one IP.

SPDY

  • Everything compressed (in HTTP headers are not compressed)
  • More files in one request
  • Supported by major browsers
