Introduction to Bootstrap

Bootstrap

  • Open Source Framework (Current version 3.3)

  • Layout and components

  • Responsive designs without boilerplate code

  • Modular and customizable

  • Easy to use styling through css classes


<button type="button" class="btn btn-success">Bootstrap is awesome!</button>

Adding Bootstrap

Boostrap is available by all major package managers

  • Bower
  • npm
  • nuget
  • direct download
  • etc.

bower install -S bootstrap
