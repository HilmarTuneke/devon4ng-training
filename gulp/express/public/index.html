Backend API documentation

Entity

The following structure is considered as single todo entity

        
{
    id: 1,
    title: 'todo 1 ',
    description: 'todo 1 description ',
    done: false
}

Operations

Route HTTP Verb Description
/services/todos GET Get all the todos.
/services/todos POST Create a todo.
/services/todos DELETE Perform archiving - remove done items.
/services/todos/:todo_id GET Get a single todo.
/services/todos/:todo_id PUT Update a todo with new data.
/services/todos/:todo_id DELETE Delete a todo.
